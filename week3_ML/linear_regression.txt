# Linear Regression: A Complete Beginner's Guide

## What is Linear Regression?

Linear Regression is one of the most fundamental and widely used techniques in machine learning. It's a **supervised learning** algorithm that helps us understand the relationship between variables by fitting a straight line through data points.

### Real-world Examples:
- Predicting house prices based on size
- Estimating salary based on years of experience
- Forecasting sales based on advertising spend

## Mathematical Foundation

### Simple Linear Regression Formula:
```
y = mx + b
```
Where:
- `y` = dependent variable (what we want to predict)
- `x` = independent variable (input feature)
- `m` = slope (how much y changes when x changes)
- `b` = y-intercept (where the line crosses the y-axis)

### The Goal:
Find the best-fitting line that minimizes the difference between actual values and predicted values.

## Types of Linear Regression

1. **Simple Linear Regression**: One input variable
2. **Multiple Linear Regression**: Multiple input variables

### 1. Simple Linear Regression Example

Let's start with a simple example predicting house prices based on size:

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score

# Create sample data
house_sizes = np.array([500, 750, 1000, 1250, 1500, 1750, 2000, 2250, 2500, 2750])
house_prices = np.array([150000, 200000, 250000, 300000, 350000, 400000, 450000, 500000, 550000, 600000])

print("House Sizes:", house_sizes)
print("House Prices:", house_prices)
```

### Visualizing the Data
```python
plt.figure(figsize=(10, 6))
plt.scatter(house_sizes, house_prices, color='blue', label='Actual Data')
plt.xlabel('House Size (sq ft)')
plt.ylabel('Price ($)')
plt.title('House Size vs Price')
plt.grid(True, alpha=0.3)
plt.legend()
plt.show()
```

### Building the Model
```python
# Reshape the data (sklearn expects 2D arrays)
X = house_sizes.reshape(-1, 1)  # Features
y = house_prices  # Target

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Get the parameters
slope = model.coef_[0]
intercept = model.intercept_

print(f"Slope (m): {slope:.2f}")
print(f"Intercept (b): {intercept:.2f}")
print(f"Equation: y = {slope:.2f}x + {intercept:.2f}")
```

### Making Predictions
```python
# Predict using the model
predicted_prices = model.predict(X)

# Plot the regression line
plt.figure(figsize=(10, 6))
plt.scatter(house_sizes, house_prices, color='blue', label='Actual Data')
plt.plot(house_sizes, predicted_prices, color='red', linewidth=2, label='Regression Line')
plt.xlabel('House Size (sq ft)')
plt.ylabel('Price ($)')
plt.title('Linear Regression: House Size vs Price')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# Predict price for a new house size
new_size = np.array([[1800]])  # Need to reshape for prediction
predicted_price = model.predict(new_size)
print(f"\nPredicted price for 1800 sq ft house: ${predicted_price[0]:,.2f}")
```

```python
# Calculate metrics

r2 = r2_score(y_test, y_pred)

print(f"\nModel Performance:")
print(f"Mean Squared Error: {mse:,.2f}")
print(f"Root Mean Squared Error: {rmse:,.2f}")
print(f"R² Score: {r2:.3f}")

# R² Score interpretation:
# 1.0 = perfect prediction
# 0.0 = no better than average
# Negative = worse than average
```

### 3. Manual Implementation (Understanding the Math)

Let's implement linear regression from scratch to understand the mathematics:

```python
class SimpleLinearRegression:
    def __init__(self):
        self.slope = 0
        self.intercept = 0
    
    def fit(self, X, y):
        # Calculate means
        x_mean = np.mean(X)
        y_mean = np.mean(y)
        
        # Calculate slope using the formula:
        # m = Σ((x - x̄)(y - ȳ)) / Σ((x - x̄)²)
        numerator = np.sum((X - x_mean) * (y - y_mean))
        denominator = np.sum((X - x_mean) ** 2)
        
        self.slope = numerator / denominator
        self.intercept = y_mean - self.slope * x_mean
    
    def predict(self, X):
        return self.slope * X + self.intercept

# Test our manual implementation

manual_model.fit(house_sizes, house_prices)

# Compare with sklearn results

print("Manual Implementation Results:")

print(f"Intercept: {manual_model.intercept:.2f}")

# Plot comparison
plt.figure(figsize=(12, 5))

plt.subplot(1, 2, 1)
```

## Key Concepts Explained

### 1. Cost Function (Mean Squared Error)
```python
def calculate_mse(actual, predicted):
    """Calculate Mean Squared Error"""
    n = len(actual)
    mse = np.sum((actual - predicted) ** 2) / n
    return mse

# Example calculation
actual = np.array([1, 2, 3, 4, 5])
predicted = np.array([1.1, 2.2, 2.8, 4.1, 4.9])
mse = calculate_mse(actual, predicted)
print(f"MSE: {mse:.4f}")
```

### 2. Gradient Descent (Conceptual Understanding)
```python
def gradient_descent_step(X, y, weights, learning_rate):
    """
    Single step of gradient descent
    This is a simplified version for understanding
    """
    n = len(y)
    
    # Make predictions
    predictions = X.dot(weights)
    
    # Calculate gradients
    dw = -(2/n) * X.T.dot(y - predictions)
    
    # Update weights
    weights = weights - learning_rate * dw
    
    return weights

# This is conceptual - sklearn handles this automatically
print("Gradient descent is how the algorithm finds the optimal parameters")
print("It iteratively adjusts the line to minimize prediction errors")
```

## Common Evaluation Metrics

```python
def evaluate_regression_model(y_true, y_pred):
    """
    Calculate common regression metrics
    """
    # Mean Absolute Error
    mae = np.mean(np.abs(y_true - y_pred))
    
    # Mean Squared Error
    mse = np.mean((y_true - y_pred) ** 2)
    
    # Root Mean Squared Error
    rmse = np.sqrt(mse)
    
    # R-squared
    ss_res = np.sum((y_true - y_pred) ** 2)
    ss_tot = np.sum((y_true - np.mean(y_true)) ** 2)
    r2 = 1 - (ss_res / ss_tot)
    
    print(f"Mean Absolute Error: {mae:,.2f}")
    print(f"Mean Squared Error: {mse:,.2f}")
    print(f"Root Mean Squared Error: {rmse:,.2f}")
    print(f"R² Score: {r2:.3f}")
    
    return {'MAE': mae, 'MSE': mse, 'RMSE': rmse, 'R2': r2}

# Example evaluation
metrics = evaluate_regression_model(house_prices, predicted_prices)
```

## Assumptions of Linear Regression

1. **Linearity**: Relationship between X and Y is linear
2. **Independence**: Observations are independent
3. **Homoscedasticity**: Constant variance of residuals
4. **Normality**: Residuals are normally distributed

## Summary

Linear Regression is:
- **Simple**: Easy to understand and implement
- **Fast**: Quick to train and predict
- **Interpretable**: Coefficients tell you the relationship strength
- **Baseline**: Good starting point for comparison

**When to use:**
- Relationship appears linear
- You need interpretability
- Quick baseline model needed
- Few features compared to samples

**Limitations:**
- Assumes linear relationships
- Sensitive to outliers
- May underfit complex patterns
